## 1. Clone this repository

```bash
$ git clone https://github.com/onleonidas/online-shopping-nest
```


## 2. Installation

```bash
$ npm install
```

## 3. Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 4. Test

```bash
# unit tests
$ npm run test
```

## 5. Swagger Api Doc
```bash
# npm run start:dev
$ and open http://localhost:3000/api-docs
````
